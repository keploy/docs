# PR Review Gate

- [ ] No blocking-severity findings
- [ ] All new files have appropriate TypeScript types when TypeScript is introduced
- [ ] No secrets introduced
- [ ] Tests exist for new utility functions when relevant; warn if missing, do not block
- [ ] No new `any` types in TypeScript additions
- [ ] No `console.log` left in production paths
- [ ] New docs pages include required frontmatter and remain discoverable in navigation
- [ ] New API or auth-related examples avoid insecure credential handling
- [ ] Impacted files have been checked and show no regressions
