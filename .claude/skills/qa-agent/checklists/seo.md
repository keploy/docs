# SEO Checklist

- [ ] No broken internal links or anchors
- [ ] Title and description are present and specific
- [ ] Canonical behavior remains correct after permalink or redirect changes
- [ ] Strategic pages have a stable social image when appropriate
- [ ] Structured data was preserved or intentionally updated
- [ ] Redirect changes do not create loops or unnecessary chains
- [ ] Sitemap, `llms.txt`, and `llms-full.txt` changes were reviewed when touched
- [ ] Navigation changes do not orphan current or versioned docs
